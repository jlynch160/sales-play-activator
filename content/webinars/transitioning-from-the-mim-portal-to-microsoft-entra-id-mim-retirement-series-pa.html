Transitioning from MIM Portal to Entra ID - MIM Retirement Part 2

MajorKey Technologies Webinar MIM Portal Self-Service Entra Migration

Summary

Part 2 of the MIM retirement series, focused specifically on replacing the MIM Portal self-service capabilities with Entra ID features. Covers self-service password reset, access request workflows, and group management as replacements for MIM Portal functions.

Key Takeaways

  • MIM Portal self-service password reset is directly replaced by Entra ID Self-Service Password Reset (SSPR) with stronger security
  • Access request workflows in MIM Portal map to Entra ID entitlement management with access packages and approval flows
  • MIM Portal group management is replaced by Entra ID self-service group management with dynamic membership and access reviews
  • Custom MIM Portal forms and approval workflows require evaluation - some map to Entra features, others need MyAccess customization
  • Migration from MIM Portal should prioritize the most-used self-service functions first to maintain user productivity during transition
