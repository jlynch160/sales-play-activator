Going Passwordless - Frictionless Secure Authentication

MajorKey Technologies Webinar Passwordless FIDO2 Phishing-Resistant MFA

Summary

Explores the journey from traditional password-based authentication to passwordless using FIDO2 security keys, Windows Hello for Business, and Microsoft Authenticator. Covers the business case for eliminating passwords, phishing resistance, and practical deployment strategies for enterprise environments.

Key Takeaways

  • Passwords are the primary attack vector in credential-based breaches - eliminating them removes the most exploited weakness
  • FIDO2 security keys provide phishing-resistant authentication by binding credentials to the physical device and the specific origin
  • Windows Hello for Business uses biometric or PIN authentication tied to the device TPM - credentials never leave the device
  • Passwordless deployment should start with privileged users and IT staff before rolling out to the broader workforce
  • Organizations can run passwordless alongside traditional MFA during transition - users gradually shift as applications support modern protocols
