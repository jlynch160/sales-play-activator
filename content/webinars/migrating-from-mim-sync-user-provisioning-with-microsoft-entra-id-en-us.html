Migrating from MIM Sync User Provisioning to Entra ID

MajorKey Technologies Webinar MIM Migration User Provisioning Entra Connect

Summary

Technical guidance for replacing Microsoft Identity Manager (MIM) Sync with Entra ID cloud-based provisioning. Covers the specific sync scenarios that can move to Entra Connect Cloud Sync, HR-driven provisioning replacements, and what to do with custom MIM extensions.

Key Takeaways

  • MIM Sync is approaching end of support and organizations need a migration plan to Entra ID cloud-native provisioning
  • Entra Connect Cloud Sync replaces most MIM Sync scenarios for AD-to-Entra synchronization with a lightweight agent
  • HR-driven provisioning from Workday and SAP can replace MIM-based HR flows with native Entra ID Governance inbound provisioning
  • Custom MIM sync rules and management agents need individual assessment - some can be replaced with Entra features, others may need HorizonID
  • Migration should be phased by sync scope - start with the simplest user populations and progressively migrate complex scenarios
