Using Entra Verified ID for Identity Proofing

MajorKey Technologies Webinar Verified ID Identity Proofing Decentralized Identity

Summary

Covers Microsoft Entra Verified ID and how it enables decentralized identity proofing for partners, customers, and employees. Explains verifiable credentials, the trust model, and practical use cases including remote employee onboarding and partner access verification.

Key Takeaways

  • Entra Verified ID uses W3C verifiable credentials to enable identity proofing without storing personal data centrally
  • Remote employee onboarding can use Verified ID to confirm identity before granting access - replacing manual document verification
  • Partner organizations can present verifiable credentials to prove their employees' identities without sharing directory data
  • Face Check adds biometric liveness detection to prevent deepfake and photo-based identity fraud during verification
  • Verified ID integrates with Entra ID conditional access so verified identity status can be required as a condition for accessing sensitive resources
